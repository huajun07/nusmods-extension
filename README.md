### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:limanjun99/nusmods-extension.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build the project

   ```sh
   npm run build
   ```

4. Follow the instructions mentioned here: https://support.google.com/chrome/a/answer/2714278?hl=en (Step 2)
